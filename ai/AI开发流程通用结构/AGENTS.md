# AI Coding 项目规范

本文档定义了 AI 在本项目中工作的核心约束，详见 `AI操作规范.md`。

---

## 目录约定

| 类型 | 路径 | 说明 |
|:-----|:-----|:-----|
| 虚拟环境 | `../venv/{项目名}-env/Scripts/python` | 项目平级目录 |
| 规范文档 | `docs/技术规范/` | 技术标准规范 |
| API 文档 | `docs/API文档/` | 接口详细设计 |
| 临时文件 | `temp/` | 用完即删 |

---

## 注意事项

- 虚拟环境命名：`{项目名}-env`，如 `myapp-env`
- AI 应根据项目结构推断工作目录
- 禁止在代码中硬编码绝对路径
