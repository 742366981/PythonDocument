# 缓存规范

本文档定义缓存操作的通用规范，适用于所有后端项目。

---

## 1. 适用范围

| 缓存类型 | 状态 | 说明 |
|:---------|:-----|:-----|
| Redis | ✅ 已支持 | 主要推荐使用 |
| Memcached | 🔜 规划中 | 预计 v2.0 支持 |

## 2. Redis缓存

### 2.1 Redis客户端

使用统一的 Redis 客户端实例：

```python
# db/redis/helpers.py

import redis
from common.settings import admin_redis_conf

pool = redis.ConnectionPool(
    host=admin_redis_conf.get('host'),
    port=admin_redis_conf.get('port'),
    password=admin_redis_conf.get('password') or None,
    db=admin_redis_conf.get('db', 0),
    decode_responses=True
)

admin_redis = redis.Redis(connection_pool=pool)
```

### 2.2 客户端方法

| 方法 | 说明 |
|:-----|:-----|
| `get(key)` | 获取值 |
| `set(key, value, ex=None)` | 设置值，ex为过期秒数 |
| `delete(*keys)` | 删除 |
| `atomic_lock(key, ex=15)` | 分布式锁（nx=True, ex指定时间） |
| `delete_pattern(pattern)` | 批量删除（支持*通配） |
| `hget/hset/hdel` | Hash操作 |

### 2.3 使用示例

```python
from common.constants import AdminRedisKeys
from db.redis.helpers import admin_redis

# 存储（运行时替换{{}}）
admin_redis.set(AdminRedisKeys.ADMIN_USER_TOKEN.format(token), user_id, ex=86400)

# 批量清除缓存
admin_redis.delete_pattern(f'{AdminRedisKeys.ADMIN_PREFIX}:cache:*')

# 分布式锁
lock = admin_redis.atomic_lock('task_lock', ex=15)
if lock:
    # 执行任务
    pass
```

## 3. Key命名规范（强制）

### 3.1 命名原则（强制）

**所有Key必须在 `common/constants.py` 的对应 Key 类中定义，禁止硬编码。**

### 3.2 Key格式

```
{前缀}:{模块}:{实体}:{标识}
```

### 3.3 Key类定义

```python
# common/constants.py

class AdminRedisKeys:
    ADMIN_PREFIX = 'app'
    ADMIN_USER_TOKEN = f'{ADMIN_PREFIX}:user:token:{{}}'
    ADMIN_USER_TOKEN_BY_ID = f'{ADMIN_PREFIX}:user:token_by_id:{{}}'
    ADMIN_CAPTCHA = f'{ADMIN_PREFIX}:captcha:{{}}'
    # ... 其他Key定义
```

### 3.4 常见Key模式

| 用途 | Key格式 | 示例 |
|:-----|:--------|:-----|
| 用户Token | `{prefix}:user:token:{token}` | `app:user:token:a1b2c3...` |
| 用户TokenByID | `{prefix}:user:token_by_id:{user_id}` | `app:user:token_by_id:123` |
| 验证码 | `{prefix}:captcha:{key}` | `app:captcha:abc123` |
| 缓存 | `{prefix}:cache:{key}` | `app:cache:user_list` |

## 4. 分布式锁（强制）

### 4.1 分布式锁实现

```python
# db/redis/helpers.py

class RedisClient:
    def atomic_lock(self, key, ex=15):
        """尝试获取分布式锁"""
        return self.client.set(key, '1', nx=True, ex=ex)

    def delete(self, *keys):
        """删除key"""
        if keys:
            self.client.delete(*keys)

    def delete_pattern(self, pattern):
        """批量删除匹配pattern的key"""
        keys = self.client.keys(pattern)
        if keys:
            self.client.delete(*keys)
```

### 4.2 定时任务分布式锁（强制）

> ⚠️ **强制要求：定时任务必须使用分布式锁**

```python
from utils.scheduler_lock import with_scheduler_lock

@with_scheduler_lock('task_name', expire_seconds=3600)
def my_task():
    """带分布式锁的定时任务"""
    # 任务逻辑...
```
