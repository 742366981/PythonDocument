# 导入导出样式规范

本文档定义导入导出文件的样式要求，适用于所有项目的Excel/CSV文件。

---

## 1. Excel模板样式

### 1.1 表头样式（强制）

| 属性 | 要求 |
|:-----|:-----|
| 字体 | 微软雅黑 / Arial |
| 字号 | 11pt |
| 字体加粗 | 是 |
| 字体颜色 | 白色（#FFFFFF） |
| 背景色 | 主色（#4472C4蓝色） |
| 文字水平居中 | 是 |
| 文字垂直居中 | 是 |
| 行高 | 25px |

### 1.2 数据行样式（强制）

| 属性 | 要求 |
|:-----|:-----|
| 字体 | 微软雅黑 / Arial |
| 字号 | 10pt |
| 字体颜色 | 黑色（#000000） |
| 文字水平居中 | 是 |
| 文字垂直居中 | 是 |
| 行高 | 22px |
| 边框 | 细线（#D9D9D9） |

### 1.3 交替行颜色（建议）

| 行类型 | 背景色 |
|:-------|:-------|
| 奇数行 | 白色（#FFFFFF） |
| 偶数行 | 浅灰（#F2F2F2） |

### 1.4 列宽要求（强制）

| 列类型 | 最小宽度 | 推荐宽度 |
|:-------|:---------|:---------|
| 编码类 | 80px | 100px |
| 名称类 | 100px | 120px |
| 是/否类 | 60px | 80px |
| 日期类 | 100px | 120px |
| 数字类 | 80px | 100px |

---

## 2. 是/否字段显示（强制）

| 导入填写 | 导出显示 |
|:---------|:---------|
| 是 | 是 |
| 否 | 否 |

导入时自动转换：`是` → 1，`否` → 0
导出时格式转换：1 → 是，0 → 否

---

## 3. 日期格式（强制）

| 格式类型 | 显示格式 | 示例 |
|:---------|:---------|:-----|
| 日期 | YYYY-MM-DD | 2024-01-01 |
| 日期时间 | YYYY-MM-DD HH:MM:SS | 2024-01-01 12:00:00 |
| 时间 | HH:MM:SS | 12:00:00 |

---

## 4. 数字格式（建议）

| 类型 | 格式 | 示例 |
|:-----|:-----|:-----|
| 整数 | 无小数 | 100 |
| 金额 | 2位小数 | 100.00 |
| 百分比 | 2位小数 | 99.99% |
| 科学计数 | 关闭 | 1000000 |

---

## 5. 模板结构（强制）

### 5.1 表头行

- 第1行为表头
- 表头必须加粗、有背景色
- 表头文字水平垂直居中

### 5.2 示例数据行（可选）

- 第2行可为示例数据（灰色填充）
- 示例数据不属于正式数据

### 5.3 数据起始行

- 正式数据从第2行（或第3行有示例时）开始
- 空行不允许存在于数据区域

---

## 6. CSV格式要求

当使用CSV格式时：

| 要求 | 说明 |
|:-----|:-----|
| 编码 | UTF-8 |
| 分隔符 | 逗号（,） |
| 引号 | 双引号（"）包裹含分隔符的字段 |
| 换行符 | LF（\n） |
| 无样式 | CSV不支持样式，忽略Excel样式要求 |

---

## 7. 错误数据样式（建议）

| 状态 | 背景色 | 字体颜色 |
|:-----|:-------|:---------|
| 正确 | 绿色（#E2EFDA） | 黑色 |
| 错误 | 红色（#FCE4D6） | 黑色 |

---

## 8. 文件命名规范（强制）

| 类型 | 命名格式 | 示例 |
|:-----|:---------|:-----|
| 导入模板 | {模块}_template.{ext} | role_template.xlsx |
| 导出文件 | {模块}_export_{日期}.{ext} | role_export_20240101.xlsx |
| CSV导入 | {模块}_template.csv | role_template.csv |
| CSV导出 | {模块}_export_{日期}.csv | role_export_20240101.csv |

---

## 9. 多Sheet要求（可选）

当一个文件包含多个Sheet时：

| 要求 | 说明 |
|:-----|:-----|
| Sheet名称 | 中文命名，反映内容 |
| 第一个Sheet | 为主数据Sheet |
| Sheet顺序 | 主数据在前，详情在后 |
