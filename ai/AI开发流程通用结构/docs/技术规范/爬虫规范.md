
**装饰器顺序**：
1. `@review_task_wrapper` - 异常处理/日志/回调
2. `@retry` - 重试机制

## 4. 任务源（任务获取）

### 4.1 任务源类型

任务获取方式可灵活扩展，常见类型：

| 类型 | 实现方式 | 示例 |
|:-----|:---------|:-----|
| Excel | 读取Excel文件 | review_query_task_args_excel() |
| 后端API | HTTP请求获取 | review_query_task_args_api() |
| 数据库 | 直接查询数据库 | review_query_task_args_db() |
| 消息队列 | 消费MQ消息 | review_query_task_args_mq() |

### 4.2 任务参数结构

每个任务包含 **执行参数** 和 **extra 上下文**：

```python
# 任务参数示例
args = (product_id, extra)  # product_id = 执行参数，extra = 上下文

# extra 上下文包含
extra = {
    'session': session,        # 可复用的Session
    'execute_time': '',        # 执行开始时间
    'complete_time': '',       # 执行结束时间
    'duration': 0,            # 执行耗时
    # ... 其他自定义字段
}
```

### 4.3 Excel任务源实现

```python
def review_query_task_args_excel():
    path = f'{BASE_DIR}/任务.xlsx'
    rows = excel_read(path)
    task_args_list = []
    for row in rows:
        product_id = str(row[0])
        if not product_id:
            continue
        extra = {}
        args = (product_id, extra)
        task_args_list.append(args)
    return task_args_list
```

### 4.4 API任务源实现

```python
def review_query_task_args_api():
    url = f'{BASE_DIR}/api/task/list'
    with requests_try('get', url) as resp:
        if type(resp) is str:
            raise GeneralError(resp)
        data = resp.json()

    task_args_list = []
    for item in data.get('data', []):
        product_id = item['product_id']
        extra = {'source': 'api'}
        args = (product_id, extra)
        task_args_list.append(args)
    return task_args_list
```

### 5.3 本地文件回调实现

```python
def review_callback_success(product_id, comment_list):
    if comment_list is None:
        return
    # 保存到JSON文件
    record_dir = f'{BASE_DIR}/review_record'
    record_path = f'{record_dir}/{product_id}.json'
    with open(record_path, 'w', encoding='utf-8') as fp:
        json.dump(comment_list, fp, ensure_ascii=False, indent=4)
```

### 5.4 后端API回调实现

```python
def review_callback_submit_api(product_id, comment_list):
    url = f'{BASE_DIR}/api/review/result'
    data = {
        'product_id': product_id,
        'comment_list': comment_list,
        'complete_time': ts_to_yyyy_mm_dd_s()
    }
    with requests_try('post', url, json=data) as resp:
        if type(resp) is str:
            raise GeneralError(resp)
        if resp.status_code != 200:
            raise GeneralError(f'回调失败：{resp.status_code}')
```

### 7.2 重试条件函数

```python
def review_retry_if_exception(e):
    hint = str(e)
    if check_hint_keywords(hint, ['代理ip失效', '接口超时', '接口连接错误', '状态码异常']):
        return True  # 可重试
    return False  # 不重试
```

### 7.3 可重试错误关键词

| 关键词 | 说明 |
|:--------|:-----|
| 代理ip失效 | 代理不可用 |
| 接口超时 | 请求超时 |
| 接口连接错误 | 连接失败 |
| HTTP错误 | HTTP请求错误 |
| 状态码异常 | 非200状态码 |

| GeneralError | 一般性异常 | 否 |

| NeedDingtalkReminderError | 需要发送提醒通知 | 温馨提示 |

### 8.2 异常安全过滤

```python
class GeneralError(Exception):
    def __str__(self):
        desc = self.args[0]
        if not PROGRAM_SECURE_SWITCH:
            return desc
        return keyword_filter(desc)  # 生产模式过滤敏感信息
```

**keyword_filter** 过滤 `获取响应失败：` 后的内容，防止敏感信息暴露。

## 10. 任务执行流程

### 10.1 任务生成器

```python
def review_task_generator():
    """
    从任务源获取任务，分发给执行器

    任务源可以是：Excel、API、数据库、消息队列等
    """
    task_args_list = review_query_task_args_list()  # 获取任务源
    for args in task_args_list:
        yield review_crawl_run, args
```

### 10.2 任务执行循环

```python
def review_run_task_loop(thread_pool):
    task_generator = review_task_generator()
    while True:
        try:
            fn, args = next(task_generator)
            thread_pool.submit(fn, *args)
        except StopIteration:
            break
    thread_pool.run_until_complete()
```

### 10.3 定时任务

```python
def review_run_timer():
    thread_size = RUN_THREAD_SIZE
    thread_pool = CustomThreadPoolExecutor(thread_size, all_completed=True)

    if IS_ONCE or IS_REVIEW_ONCE:
        review_run_task_loop(thread_pool)
        return

    while True:
        timer = Timer(seconds, review_run_task_loop, args=(thread_pool,))
        timer.start()
        timer.join()
```

## 11. 任务包装器

### 11.1 包装器职责

```python
def review_task_wrapper(func):
    @wraps(func)
    def wrapper(*args, **kwargs):
        extra = args[-1]  # 获取extra参数
        try:
            # 1. 记录执行时间
            extra['execute_time'] = ts_to_yyyy_mm_dd_s()

            # 2. 执行任务
            result = func(*args, **kwargs)

            # 3. 记录完成时间
            extra['complete_time'] = ts_to_yyyy_mm_dd_s()
            extra['duration'] = yyyy_mm_dd_s_to_ts(extra['complete_time']) - yyyy_mm_dd_s_to_ts(extra['execute_time'])

            # 4. 执行回调
            extra['callback_execute_time'] = ts_to_yyyy_mm_dd_s()
            review_callback_success(*result)
            extra['callback_complete_time'] = ts_to_yyyy_mm_dd_s()
            extra['callback_duration'] = yyyy_mm_dd_s_to_ts(extra['callback_complete_time']) - yyyy_mm_dd_s_to_ts(extra['callback_execute_time'])

            # 5. 记录结果日志
            msg = f'任务类型：{REVIEW_TASK_TITLE}，参数：{args}，执行完毕，爬虫耗时Xs，回调耗时Ys'
            review_result_log.save_info(msg)
            return result

        except Exception as e:
            # 6. 分类处理异常
            if isinstance(e, GeneralError):
                review_error_log.save_error(...)
            elif isinstance(e, NeedDingtalkFailError):
                fail_dingtalk.send_text_msg(...)
            elif isinstance(e, NeedDingtalkReminderError):
                reminder_dingtalk.send_text_msg(...)
            else:
                review_error_log.save_critical(traceback_str)
            return hint
    return wrapper
```

## 16. Redis客户端

### 16.1 连接池实现

**必须使用连接池**，禁止直接创建 Redis 连接。

```python
# db/redis/helpers.py

class Redis:
    def __init__(self, host, port, password, db=0):
        self.pool = redis.ConnectionPool(
            host=host,
            port=port,
            password=password,
            db=db,
            decode_responses=True
        )
        self.client = redis.StrictRedis(connection_pool=self.pool)
```
